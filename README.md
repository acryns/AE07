# AE07
Cupid - als ultimative Datingapp in die Entwicklung gegangen - als nicht-so-richtig-toll funktionierendes Adressbuch "beendet". 
Der - eigens gelegte - Fokus lag auf der (optionalen) Anbindung an eine Datenbank. Die GUI ist Mittel zum Zweck und stellt in drei 
Tabs die gespeicherten Daten dar: Als Liste, Einträge im Detail, Neuanlage.
Das funktioniert alles nur so halbgut bisher und sieht fürchterlich aus.
